# Recommended Image Sizes for Campaign 2028

## Candidate Photos
- **Dimensions**: 400x400 pixels (square)
- **Format**: JPG
- **Location**: `images/[candidate-id].jpg`
- **Examples**: 
  - `images/harris.jpg`
  - `images/desantis.jpg`
  - `images/yang.jpg`

## Party Logos/Photos
- **Dimensions**: 200x200 pixels (square) or 300x150 pixels (rectangular)
- **Format**: PNG with transparency or JPG
- **Location**: `images/party-[code].png` or similar
- **Note**: Currently party identification is handled through color coding in the UI

## VP Photos
- **Dimensions**: 400x400 pixels (square)
- **Format**: JPG
- **Location**: `images/[vp-id].jpg`
- **Examples**:
  - `images/shapiro.jpg`
  - `images/rubio.jpg`

## Placeholder Image
- **File**: `images/scenario.jpg`
- **Dimensions**: 400x400 pixels
- **Purpose**: Used as fallback when candidate-specific image is not available

## Notes
- Keep file sizes under 200KB for optimal loading
- Use consistent aspect ratios for a professional appearance
- High quality headshots work best
- Ensure good contrast for visibility on dark backgrounds
